# projects

A small collection of scripts I'm using to learn how tools like Nmap work under the hood, how port scanning works, and how to automate tasks with Bash.

## What's in here

- **`nmapshell.sh`** — Bash script for learning Nmap-style scanning and automation.
- **`webrecon.sh`** — Bash script for basic web recon automation.
- **`main.py`** — Python script for learning port scanning basics.
- **`banner_grabber.py`** — Python script for learning how banner grabbing works (identifying services running on open ports).

## Why

This repo exists mainly as a learning exercise — understanding how common security tools work by building simplified versions myself, and getting more comfortable with Bash scripting and automation along the way.

## Note

These scripts are for educational purposes. Only use them against systems you own or have explicit permission to test.
